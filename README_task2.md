# Task 2: Cross-Attention Visualization for Entailment Reasoning

## Problem Statement
This task audits visual-semantic reasoning by exposing where fusion attention focuses for entailment decisions.

Project baseline for all Task 1-5 runs:
- Backbone checkpoint: `final_sota_visual_entailment3.pth` (clean-test ~64.18%)
- Chosen over a separate 73.7% validation checkpoint due to better custom-data behavior.

Primary goals:
- Add explicit cross-attention between image patches and text tokens in fusion.
- Hook attention tensors using `register_forward_hook()`.
- Curate 20 examples across entailment/neutral/contradiction with mixed difficulty.
- Visualize token-conditioned and overall heatmaps on original images.

## Pipeline
1. Load `cleaned_snli_ve_test.csv` and checkpoint `final_sota_visual_entailment3.pth`.
2. Build `VisualEntailmentWithFusionAttention` with 2 fusion-attention layers and 8 heads.
3. Run prediction pool generation (`pool_size=2500`) and compute difficulty labels.
4. Select 20 examples with class/difficulty coverage.
5. Register hooks on fusion-attention layers and save raw tensors (`.npy`).
6. Generate token overlays and class-gallery artifacts.
7. Save summary and tensor index for reproducibility.
